# 灵感库浏览器插件 v5.3.1

## 本次修复

- 修复 `background.js` 的语法错误，使 Manifest V3 Service Worker 可正常启动。
- 右键菜单从 `image/video` 扩展到 `image/video/link/page`，支持图片上方有链接、按钮或透明遮罩的网站。
- Pinterest：从 `srcset` 选最高分辨率，并尝试将缩略图路径升级为 `/originals/`。
- 花瓣：识别 `hbimg.huaban.com` CDN，移除 `imageView2` 缩放参数。
- 花瓣会拦截浏览器原生右键菜单，因此 v5.3.1 在花瓣图片上增加了「保存到灵感库」悬浮按钮，不再依赖右键入口。
- 小红书：识别 `xhscdn.com` 图片键，重建原图地址；当视频标签是 `blob:` 时，从已加载资源和页面内嵌状态中寻找真实视频 CDN 地址。
- 新增来源页 MAIN world 下载和 content script 后备下载，保留来源站 Cookie/Referer。
- 下载或上传失败时明确报错，不再把临时外链当作已转存文件写入数据库。

## 安装测试

1. 打开 `chrome://extensions/`。
2. 开启「开发者模式」。
3. 选择「加载已解压的扩展程序」，指向 `src/moodboard-extension`。
4. 分别在 Pinterest、花瓣、小红书打开单张详情页，把鼠标放在主图或视频画面上右键。
5. 点击「保存到灵感库 📌」。

## 已知边界

- 当前适合单张图片和中小视频。大于 80MB 的视频不经过扩展消息通道，需要另外增加服务端「URL 转存到 Appwrite Storage」接口。
- 小红书的页面签名和数据结构会变化。本版优先复用页面已加载资源，没有复制易失效的私有签名算法。
