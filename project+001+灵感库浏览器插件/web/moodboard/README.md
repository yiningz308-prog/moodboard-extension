# 个人灵感库

单用户、本地优先的视觉素材管理网页，配合 Chrome 插件采集小红书、花瓣、Pinterest 等网页中的图片、GIF 和视频。

## 数据放在哪里

- 原始文件和元数据存入浏览器 IndexedDB，不上传公司服务，也不需要登录。
- 上传图片时串行生成最长边 720px 的 WebP 预览图；图库只显示预览图，详情和下载使用原始文件。
- 首屏最多渲染 30 项，滚动后继续加载；图片使用浏览器懒加载。
- 浏览器本地库不是跨设备云盘。请定期使用“导出”生成 `manifest.json + media/` 完整备份，在新设备用“导入”恢复。

## 直接启动

双击项目根目录的 `启动个人灵感库.command`，浏览器打开 `http://localhost:5173`。保持终端窗口运行即可。

## 开发与构建

```bash
pnpm install
pnpm dev
pnpm build
```

生产文件位于 `dist/`，可以放在任何静态网站服务上。无论部署在哪，当前版本的数据仍保存在访问该域名的本机浏览器内。

## 浏览器插件

在 Chrome 扩展程序页加载：

`src/moodboard-extension`

插件默认连接 `http://localhost:5173`。如部署到自己的域名，请打开插件“选项”，修改“灵感库网页地址”。

## 自动跨设备同步

当前版本采用手动导出/导入，完全不需要阿里云或腾讯云。如果要自动同步，需要增加：对象存储、元数据同步接口、设备身份验证和冲突处理。静态前端不能安全保存 OSS/COS 的永久密钥。
